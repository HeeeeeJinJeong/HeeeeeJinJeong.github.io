# Django
## 파이썬 프로젝트 만들기
### 장고 설치 : pip install django
### 장고 프로젝트 만들기 : django-admin startproject config .
### DB 초기화 : python manage.py migrate
### 관리자 계정 생성 : python manage.py createsuperuser
- pw 변경 : python manage.py changepassword admin
